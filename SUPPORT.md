### Common Questions

**Q: Why is my data delayed?**
A: Default free data sources provide delayed data (typically 15-20 minutes). For real-time data, connect an optional authenticated provider.

**Q: How do I add a new data source?**
A: See [CONTRIBUTING.md](./CONTRIBUTING.md) for instructions on creating data adapters.

**Q: Can I use this commercially?**
A: Yes! The project is MIT licensed. However, check the terms of each data source you use.

**Q: Is this affiliated with Bloomberg?**
A: No. This is an independent open-source project. Bloomberg Terminal¬Æ is a registered trademark of Bloomberg Finance L.P.

## üêõ Bug Reports

### Before Reporting

1. **Update** to the latest version
2. **Search** [existing issues](https://github.com/borealBytes/open-fin-terminal/issues)
3. **Check** if it's a known issue

### How to Report

1. Use the [Bug Report template](.github/ISSUE_TEMPLATE/bug_report.md)
2. Provide:
   - Clear description
   - Steps to reproduce
   - Expected vs actual behavior
   - Environment (browser, OS, version)
   - Console errors (if any)
   - Screenshots (if relevant)
3. Use a descriptive title
